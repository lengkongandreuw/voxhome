# VoxHome Landing Page

Static landing page for VoxHome, built for HVAC, plumbing, and roofing contractors in the USA market.

## Files

- `index.html` - complete landing page with inline CSS
- `assets/voxhome-hero.png` - hero image asset

## Publish to GitHub Pages

1. Create a new GitHub repository, for example `voxhome-landing-page`.
2. Upload `index.html` and the `assets` folder to the repository root.
3. Open repository `Settings` > `Pages`.
4. Set source to `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Save, then wait for GitHub Pages to publish the site.
